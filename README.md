# 小倉GALAXY - KOKURA GALAXY -

## 🌌 プロジェクト概要

「小倉GALAXY」は、北九州市小倉に建設予定（架空）の超高層分譲マンションのプロモーションサイト制作プロジェクトです。
既存のマンションデベロッパーサイト（例：Century Mansion）の堅実なレイアウト構成をベンチマークとしつつ、「宇宙空間」「近未来的ラグジュアリー」という大胆なテーマを融合させた実験的なコンセプトモデルです。

本プロジェクトは、フレームワークに依存せず、HTML/CSS/JavaScriptの基礎技術（Vanilla JS）のみを用いて、高パフォーマンスかつ没入感のあるWeb体験（Immersive Web Experience）を構築することを主眼としています。不動産サイトにありがちな情報の羅列を避け、ブランドの世界観を視覚的・直感的に伝えるストーリーテリング型の手法を採用しました。

## 📸 デザインコンセプト & ブランドイメージ

**テーマ：「宇宙（そら）に一番近い場所、小倉。」**

北九州の夜景と星空を独占する、地上40階建てのタワーレジデンスという設定です。
従来の高級マンション広告に見られる「重厚感」「伝統」というキーワードに加え、「静寂」「無限」「革新」を核となるコンセプトに据えました。Webサイト全体を一つの宇宙空間（Universe）に見立て、ユーザーがスクロールする行為を「宇宙への旅」とリンクさせています。

## 🎨 カラーパレット戦略

色彩心理学に基づき、ラグジュアリーさと先進性を両立させる配色計画を行いました。

- **Primary Dark (宇宙の背景):** `#050a14`
  - 完全な黒（`#000000`）ではなく、わずかに青みを含んだダークネイビーを採用。ディスプレイ上で奥行きと深遠さを感じさせ、長時間閲覧しても目が疲れにくい効果を狙っています。
- **Secondary Dark (セクション背景):** `#0b1628`
  - セクションごとの区切りを明確にしつつ、グラデーションのような自然な繋がりを持たせるためのディープブルーです。
- **Accent Gold (星・高級感):** `#cfaa6e`
  - ギラつきを抑えたシャンパンゴールド。近未来的な空間の中に、邸宅としての温かみとステータス性を付与するアクセントカラーとして機能します。
- **Text White (本文):** `#f0f0f5`
  - 純白ではなく、背景色と馴染みの良いブルーホワイトを採用し、高いコントラスト比を維持しながら視認性を確保しています。
- **Text Gray (補足):** `#aab2c0`
  - 情報の階層構造を視覚的に整理するためのサブカラーです。

## 🖋 タイポグラフィ設計

- **見出し (和文):** Zen Old Mincho または Noto Serif JP
  - 明朝体を採用することで、日本の伝統的な美意識と高級マンションとしての品格を演出します。

  - 古代ローマの碑文にインスパイアされたセリフ体。その幾何学的かつ古典的な形状は、「時を超えた価値」と「銀河的な壮大さ」の両方を表現するのに最適です。
- **本文:** Noto Serif JP
  - 可読性とWebフォントとしての読み込み速度のバランスを考慮し、システムフォントとの親和性も高いNotoシリーズを選定しました。

## 🏗 UX/UI コンテンツ仕様

### 1. Header (Global Navigation)
- **仕様:** `position: fixed` による追従型ヘッダー。
- **デザイン:** 背景には `backdrop-filter: blur(10px)` を適用し、磨りガラスのような質感を表現。背面のコンテンツ（星空）を完全に隠さず、透過させることで空間の広がりを阻害しません。
- **インタラクション:** 各リンクにホバー時、ゴールドのアンダーラインが左から右へ伸びるマイクロインタラクションを実装し、繊細な高級感を演出します。

### 2. Hero Section (First View)
- **役割:** サイト訪問時の最初の3秒でユーザーの心を掴む（アトラクト）。
- **視覚効果:** 宇宙ステーションや近未来都市を想起させる建築画像を配置。CSS Keyframesアニメーションにより、画像全体が呼吸するようにゆっくりと拡大・縮小を繰り返し、静止画でありながら動的な生命感を与えます。
- **演出:** ページロード完了と同時に、キャッチコピーが下からふわっと浮き上がる（Fade-in Up）演出を加え、物語の始まりを印象付けます。

### 3. Concept Section
- **レイアウト:** 日本語特有の美しさを活かすため、`writing-mode: vertical-rl` を使用した縦書きレイアウトを採用。
- **構成:** 左側に縦書きのコピー、右側にラウンジ画像を配置するアシンメトリーな構成（スマホ時は縦積み）。余白（Negative Space）を多めに取ることで、ラグジュアリーな「余裕」を表現しています。
- **背景処理:** Heroセクションからの急激な変化を避けるため、線形グラデーションを用いて視線を自然に誘導します。

### 4. Location Section (Access)
- **情報設計:** 単なる地図情報の羅列ではなく、「小倉駅徒歩5分」という圧倒的な利便性がもたらすライフスタイルを提案。
- **UIデザイン:** 黒背景にゴールドの細いボーダーラインや「✦」アイコンを使用し、会員制クラブの招待状のような特別感を演出。
- **可視化:** 主要都市へのアクセス時間をリスト化し、ビジネスエグゼクティブへの訴求力を高めます。

### 5. Design Section (Gallery)
- **レイアウト:** `display: grid` を用いた3カラムのモダンなグリッドレイアウト。
- **インタラクション:** カード型レイアウトにマウスオーバー（Hover）することで、以下の複合アニメーションが発生します。
    - 画像がわずかに拡大（Zoom-in）。
    - 画像の上に半透明の黒グラデーションが重なる。
    - 英語のキャプション（例：SKY LOUNGE）が下からスライドイン。
    - この一連の動きにより、ユーザーの操作に対するフィードバックを明確にし、回遊性を高めます。

### 6. Footer
- **内容:** ブランドロゴ、著作権表記、および本サイトがフィクションである旨の注記。信頼感を損なわないよう、シンプルかつ整然としたレイアウトで締めくくります。

## ✨ アニメーション技術詳細

### A. 背景パーティクル (HTML5 Canvas API)
Webサイト全体を覆う「星空」は、静止画ではなくJavaScriptによる動的な描画で生成されています。
- **ロジック:**
    - 画面解像度に応じて星の生成数を動的に計算（密度調整）。
    - 各粒子（星）に対し、X/Y座標、サイズ、透明度、移動速度をランダムに割り当て。
    - `requestAnimationFrame` メソッドを使用して描画ループを作成し、60fpsのスムーズなアニメーションを実現。
- **挙動:** すべての星がゆっくりと上方向（Y軸マイナス方向）へ移動することで、上昇気流や無重力空間の浮遊感を表現しています。また、確率的に透明度を変更することで「星の瞬き」をシミュレートしています。

### B. スクロール連動フェードイン (Intersection Observer API)
従来の `scroll` イベントリスナーはパフォーマンス負荷が高いため、より現代的で高効率な Intersection Observer API を採用しました。
- **実装:**
    - 監視対象の要素（タイトル、画像、テキスト）に `.fade-in-up` クラスを付与し、初期状態では `opacity: 0`、`transform: translateY(30px)` で隠しておきます。
    - 要素がビューポート（画面）の10%に入った瞬間を検知し、`.visible` クラスを付与。
    - CSSの `transition` プロパティにより、要素が元の位置に戻りながら不透明になるアニメーションが発火します。

## 🗓 開発ロードマップ

### Phase 1: 構造化とセマンティクス (HTML)
- プロジェクトディレクトリのセットアップ。
- SEOとアクセシビリティを意識したHTML5マークアップ（`<header>`, `<main>`, `<section>`, `<article>`, `<footer>`タグの適切な使用）。
- 各セクションへのダミーテキストとプレースホルダー画像の配置。

### Phase 2: デザインシステムの構築 (CSS)
- `reset.css` によるブラウザ間スタイルの統一。
- CSSカスタムプロパティ（変数）の定義による、色やフォントの一元管理。
- FlexboxとGrid Layoutを用いたレスポンシブな骨組みの作成。
- 縦書きテキストなど、日本語サイト特有のスタイリング実装。

### Phase 3: インタラクションの実装 (JavaScript)
- Canvasを用いた背景パーティクルシステムの開発。
- Intersection Observerを用いたスクロール検知ロジックの実装。
- パフォーマンスチューニング（リサイズイベントの最適化など）。

### Phase 4: ブラッシュアップと最適化
- **レスポンシブ対応:** メディアクエリ（`@media`）を使用し、スマートフォン、タブレット、デスクトップでの表示崩れを修正。特にナビゲーションやグリッドの挙動を調整。
- **アセットの差し替え:** 高画質な著作権フリー画像（Unsplash等）への差し替えと、画像圧縮による読み込み速度の向上。
- **コード品質:** インデントの整理、コメントの記述、不要なコードの削除。

## 🛠 使用技術スタック

本プロジェクトでは、Web標準技術への深い理解を示すため、あえてjQueryやReactなどのライブラリ・フレームワークを使用していません。

- **HTML5:** セマンティックWeb、Canvas API
- **CSS3:** Variables, Flexbox, Grid, Keyframes Animation, Backdrop-filter
- **JavaScript (ES6+):** Class構文 (想定), Arrow Functions, DOM Manipulation, Intersection Observer API
- **Git / GitHub:** バージョン管理
